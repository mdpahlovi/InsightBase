# Insight Base

A modern AI-powered tool that enables users to create, manage, search, and summarize articles.

## Run Locally

Clone the project

```bash
  git clone https://github.com/mdpahlovi/InsightBase
```

Go to the project directory

```bash
  cd InsightBase
```

Run with docker

```bash
  docker compose up -d
```
